Web Development

CA120 WEB DEVELOPMENT (38 Contact Hours--2 Credits) In this course students learn to use HTML, CSS, and JavaScript to develop welldesigned Web pages. Students learn to apply appropriate techniques and to include forms, images, and tables.

5 Things I learned in Class

  • I learned what HTML is used for and the benefits of using HTML.
  • I learned some of the basic symbols and functions of HTML
  • I learned how to change the title and font size of my HTML pages
  • I learned how to format my HTML page
  • I learned how to make tables, lists, links, and how to put videos or pictures in my HTML files
